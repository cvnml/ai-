# Claude Skills

This repository stores Claude Code skills and routing helpers.

## Included bundle

- `atomic-plan-executor`: legacy launcher that routes to split skills.
- `plan-mode-plus`: enhanced planning flow.
- `modular-plan-docs`: approved plan persistence flow.
- `plan-implementation-runner`: mature approved plan implementation runner.
- `edit-node-blueprint`: per-write-node blueprint confirmation.
- `execution-checkpoint`: lightweight execution status checkpoint.
- `implementation-review`: implementation completeness review.
- `frontend-test`: frontend testing design and execution helper.
- `skill-mining-review`: reusable Skill mining review flow.
